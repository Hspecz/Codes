//==============================
// Riggo No Dispell Item Bonus Procedure
//==============================
// Instructions
// Apply the changes below to your src/ files.
// Recompile your server.
// Add the constant to your db/const.txt.
// Use it in an item: { bonus bNoDispell; }.

1. Open pc.cpp
	Find case SP_CRT
	Paste after 
		/// bNoDispell Mod
		case SP_NODISPELL: // 
			if (sd->state.lr_flag != 2)
				sd->bonus.no_dispell = 1;
			break;
		// bNoDispell Mod
2. Open script.hpp
	On the very first line after the Rathena copyright insert this
	// Custom bNoDispell ID
	#define SP_NODISPELL 100001
	// Custom bNoDispell ID
3. Open script_constants.hpp
	Find "export_constant2("bWeaponSubSize", SP_WEAPON_SUBSIZE);"
insert after the last constant export entry
	// bNoDispell Mod
	export_constant2("bNoDispell", SP_NODISPELL);
	// bNoDispell Mod
4. How to Use
Create an item (like a custom card or headgear) and add the script:
{ bonus bNoDispell,1; }

When a player equips this item, any Sage casting Dispell on them will fail, but other skills like Decrease Agi or Divest will still work.
